🌐 Personal Portfolio Website

📌 Overview

This project is a responsive Personal Portfolio Website developed using HTML, CSS, and JavaScript. It showcases my profile, skills, projects, education, and contact information in a clean and modern layout.

✨ Features

- Responsive design for desktop and mobile devices
- Professional Home section
- About Me section
- Skills section
- Projects showcase
- Contact section
- Smooth scrolling navigation
- Simple and user-friendly interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click "index.html" or open it using Live Server in VS Code.
4. Explore the portfolio website in your browser.
